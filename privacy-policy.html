Privacy Policy — Ojisan Romance

Last updated: 2026-07-17

Overview

This application ("the App") is a personal, single-user tool built and operated solely by its developer to publish the developer's own AI-generated short fiction (illustrated text posts) to the developer's own Instagram and Threads accounts. The App is not distributed to, or used by, any other person or business, and it does not act on behalf of any third-party Instagram or Facebook account.

What data the App accesses

When authorized by the developer's own Instagram Professional account, the App may access:

  • Basic account info (Instagram user ID, username) needed to identify the target account for publishing.
  • The ability to create and publish media containers (images and captions) to that same account.

The App does not read, collect, or store data belonging to any other Instagram user, follower, or third party. It does not access private messages, comments from other users, or any analytics/insights data.

How the data is used

  • The access token is used only to publish content (image + caption) that the developer has authored, to the developer's own account.
  • No data obtained via the Instagram/Threads API is shared with, sold to, or disclosed to any third party.
  • No data is used for advertising, profiling, or any purpose other than publishing the developer's own posts.

Data storage and retention

  • The access token is stored locally on the developer's own machine (as an environment variable / local file), never transmitted to any third-party server.
  • No Instagram/Threads user data is stored in any database. The App only reads locally-generated story text and images (authored by the developer) to construct posts.

Data deletion

The developer can revoke the App's access at any time via Instagram/Facebook account settings ("Apps and websites" → remove access), which immediately invalidates the App's access token. Because no user data is stored by the App itself, there is no separate data-deletion process required.
